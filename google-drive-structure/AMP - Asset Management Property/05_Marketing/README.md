- Assets/ (images, videos)

### Reports/
Marketing performance reports:
- Weekly/ (quick wins, issues)
- Monthly/ (comprehensive analysis)
- Quarterly/ (strategic review)

## Workflow

### Creating New Ad Campaign

1. Define campaign goals and target audience
2. Create ad creatives in appropriate platform folder
3. Schedule content in Content_Calendar/
4. Launch campaign
5. Track performance in Reports/
6. Optimize based on results

### Content Creation

1. Plan in Content_Calendar/
2. Create using Brand_Assets/Templates/
3. Save in platform-specific folders
4. Move to Scheduled/ when ready
5. After publishing ‚Üí Move to Published/

## File Naming

**Ad Creatives:**
`[Platform]_[Type]_[Campaign]_[Date]_v[Version]`

Examples:
- FB_Feed_Post_Jomtien_Condo_2026-01-20_v1.jpg
- Google_Display_Banner_Villa_Sale_2026-01-v2.png
- IG_Story_New_Listing_2026-01-21.jpg

**Reports:**
`Marketing_Report_[Period]_[Date].xlsx`

Examples:
- Marketing_Report_Weekly_2026-01-W4.xlsx
- Marketing_Report_Monthly_2026-01.xlsx

## Tips

- **Consistency is key** - Use brand assets for all materials
- **Test everything** - A/B test ad copy, images, CTAs
- **Track performance** - Data-driven decisions only
- **Archive old materials** - Keep your folders clean
- **Follow platform guidelines** - Each platform has specific requirements

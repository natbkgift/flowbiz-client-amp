## Using SOPs

### When to Use
- New team member training
- Process refresh/review
- When unsure about procedure
- Quality control reference

### How to Use
1. **Read completely** before starting
2. **Follow step-by-step** - don't skip
3. **Check off items** as you go
4. **Ask questions** if unclear
5. **Suggest improvements** if process could be better

### Updating SOPs
- **Quarterly review** - Are processes still current?
- **Version control** - Keep old versions in archive
- **Team input** - Get feedback from people using them
- **Update date** - Always date the latest version

## Checklists Best Practices

- **Print and use** - Physical checklists prevent mistakes
- **Check off items** - Don't trust memory
- **Keep handy** - Have copies ready
- **Update regularly** - Add items as you learn

## Tips

- **SOPs save time** - Create once, use forever
- **Checklists prevent errors** - Use them every time
- **Document everything** - If it's not written, it doesn't exist
- **Keep it simple** - Clear, concise, actionable
- **Review regularly** - Processes should improve over time

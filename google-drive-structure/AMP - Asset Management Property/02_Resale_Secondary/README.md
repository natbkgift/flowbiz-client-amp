## Overview

This folder manages all resale and secondary market property listings - properties that are not new projects from developers.

## Main Files

**Resale_Master_List.xlsx** (Create this file)
- Central database of all resale properties
- Include: Property ID, Location, Type, Price, Owner Contact, Status
- Update regularly

## Structure

**Property_Details/**
- Create a folder for each property using Property_ID
- Each property folder should contain:
  - Property_Sheet.docx (detailed information)
  - Photos/ (property photos)
  - Documents/ (ownership docs, title deeds)
  - Valuations/ (appraisal reports)

**_Archive/**
- Move sold properties here
- Keep for 1 year minimum for reference

## Workflow

1. New resale listing ‚Üí Add to Resale_Master_List.xlsx
2. Create folder in Property_Details/[Property_ID]/
3. Upload photos and documents
4. Update status in master list as it progresses
5. When sold ‚Üí Move to _Archive/

## Property ID Format

Use format: `RS-[YYYY]-[Number]`

Examples:
- RS-2026-001
- RS-2026-002

## Tips

- Take high-quality photos for resale listings
- Get valuation reports before pricing
- Keep owner contact info confidential
- Regular follow-ups on listings

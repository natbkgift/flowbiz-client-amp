## Workflow

### Using Contract Templates

1. **Make a copy** of the template (NEVER edit original)
2. **Rename** to: `[Type]_[Property]_[Client]_[Date].docx`
3. **Fill in** all required fields
4. **Review** carefully before sending to client
5. **Get reviewed** by legal (if required)

### Storing Executed Contracts

1. **Scan** all signed documents (high quality PDF)
2. **Create folder** in Executed_Contracts/[Type]/[Year]/[Property_ID]/
3. **Upload** contract + supporting documents
4. **Keep original** in physical safe
5. **Backup** (automatic via Google Drive + manual quarterly)

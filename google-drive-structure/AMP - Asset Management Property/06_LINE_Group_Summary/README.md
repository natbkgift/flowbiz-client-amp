## Overview

This folder tracks daily property listings and discussions from LINE groups. Many properties in Thailand are shared via LINE groups, and this folder helps organize that information.

## Main Files

**Daily_Summary_[YYYY-MM].xlsx** (Create for current month)
- Daily log of properties shared in LINE groups
- Include: Date, Property Type, Location, Price, Contact, Group Name
- One file per month

## Structure

**Monthly_Archives/**
- Store completed monthly summaries
- Keep indefinitely for reference
- Format: `Summary_[YYYY-MM].xlsx`

**Screenshots/**
- Organize by date: [YYYY-MM-DD]/
- Save screenshots of property postings
- Include contact info and details
- Name files clearly: `Property_Type_Location_01.jpg`

**Processed/**
- Properties that have been added to master lists
- Properties that have been followed up
- Helps avoid duplicates

## Workflow

### Daily Routine

1. **Morning Review** (30 minutes)
   - Check LINE groups for new property postings
   - Take screenshots of interesting properties

2. **Documentation** (15 minutes)
   - Add to Daily_Summary spreadsheet
   - Save screenshots in Screenshots/[Today's Date]/
   - Note key details

3. **Follow-up** (As needed)
   - Contact property owners/agents
   - Add verified properties to master lists
   - Move processed items to Processed/

### End of Month

1. Close current month's Daily_Summary
2. Move to Monthly_Archives/
3. Create new Daily_Summary for next month
4. Clean up Screenshots/ folder (compress or delete low-quality)

## Daily Summary Spreadsheet Columns

Suggested columns:
- Date
- Time Posted
- Property Type (Condo/Villa/House/Land)
- Location
- Price Range
- Bedrooms/Size
- Contact Name
- Contact Phone
- LINE Group Name
- Screenshot File
- Status (New/Contacted/Added to System/Not Interested)
- Notes

## Screenshot Guidelines

- **Quality:** Clear, readable text
- **Complete info:** Capture full property details and contact
- **Naming:** `[Property_Type]_[Location]_[Number].jpg`
- **Organization:** Use date folders

## Tips

- **Check daily** - Good properties go fast
- **Act quickly** - Contact promising leads within hours
- **Avoid duplicates** - Check Processed/ before following up
- **Build relationships** - Many LINE group members are repeat posters
- **Keep organized** - Screenshots pile up fast

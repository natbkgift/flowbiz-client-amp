## Usage

1. **Copy this entire folder**
2. **Rename** to your property name (e.g., `Jomtien_Beach_Condo`)
3. **Delete this README** after copying
4. **Fill in all sections** below

## Required Files

Create these files in the root of your property folder:

- [ ] `Property_Info.docx` - Complete property details
- [ ] `Pricing_Details.xlsx` - Unit pricing and availability
- [ ] `Brochure.pdf` - Official project brochure (if available)

## Photo Guidelines

- **Format:** JPG for photos, PNG for graphics
- **Resolution:** Minimum 1920x1080 for main photos
- **File size:** Compress to under 2MB per photo
- **Naming:** Descriptive_Location_Number.jpg

## Tips

- Keep original files in Raw_Footage or a backup location
- Update Property_Info when details change
- Mark unavailable units in Pricing_Details
- Archive when project is sold out

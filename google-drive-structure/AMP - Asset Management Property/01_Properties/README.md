## Structure

This folder contains all new property projects organized by type:

- **Condo/** - Condominium projects
- **Villa/** - Villa projects
- **House/** - House projects
- **Land/** - Land plots
- **_Archive/** - Archived/sold projects

## Adding a New Property

1. Navigate to the appropriate property type folder (Condo/Villa/House/Land)
2. Copy the `_Template` folder
3. Rename to your property name (e.g., `Jomtien_Beach_Condo`)
4. Fill in all required documents and photos
5. Use the photo subfolder structure (Exterior, Common Areas, Units, Views)

## Property Folder Structure

Each property should have:
- Property_Info document
- Pricing_Details spreadsheet
- Brochure (if available)
- Photos/ (organized by type)
- Videos/ (if available)
- Floor_Plans/ (PDF files)
- Documents/ (contracts, specs)
- Marketing/ (ad copy, social media content)

## Naming Convention

- Use underscores for spaces: `Project_Name`
- Keep under 50 characters
- Be descriptive: `Luxury_Beachfront_Condo_Pattaya`

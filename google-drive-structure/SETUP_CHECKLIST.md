## Common Issues & Solutions

### Issue: Can't find file
**Solution:**
- Check _Archive folders
- Use Google Drive search
- Check if in wrong folder
- Ask team if moved

### Issue: Duplicate files
**Solution:**
- Check naming convention
- Delete duplicates
- Keep only latest version
- Update links if any

### Issue: Wrong permissions
**Solution:**
- Review permission table in GOOGLE_DRIVE_STRUCTURE.md
- Update folder sharing settings
- Remove and re-add if needed

### Issue: Running out of space
**Solution:**
- Archive old files
- Compress images
- Delete unnecessary videos
- Upgrade storage plan if needed

## Troubleshooting

### Common Issues & Solutions

**Issue: High CPC (> 50 THB)**

Solutions:
- Add more negative keywords
- Improve quality score (better landing pages)
- Try different match types
- Target less competitive keywords
- Lower bids, accept less traffic

**Issue: Low CTR (< 1%)**

Solutions:
- Rewrite ads with stronger CTAs
- Include prices in ads
- Add emotional triggers
- Use ad extensions
- Ensure ad relevance to keywords

**Issue: Low Conversion Rate (< 2%)**

Solutions:
- Review landing page (is it relevant?)
- Simplify contact form (fewer fields)
- Add trust signals (reviews, testimonials)
- Make contact options more prominent
- Check mobile experience
- Add live chat/WhatsApp
- Speed up page load time

**Issue: Leads but no quality**

Solutions:
- Add qualifying questions in form
- Be clearer about pricing on page
- Target more specific keywords
- Review ad copy (setting wrong expectations?)
- Improve lead nurturing process

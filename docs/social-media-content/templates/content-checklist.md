## General Content Checklist

Use this checklist for EVERY post before publishing.

### Content Strategy Alignment
- [ ] Aligns with monthly content calendar
- [ ] Supports current marketing objectives
- [ ] Fits target audience profile
- [ ] Appropriate for property type (New Projects/Resale/Rental/Branding)
- [ ] Matches content type goal (Listing/Educational/Lifestyle/Promotional)
- [ ] Timing is optimal for target audience

### Information Accuracy
- [ ] All property details verified (price, size, location)
- [ ] Contact information is current and correct
- [ ] Links are working and direct to correct pages
- [ ] Property availability confirmed (not sold/rented)
- [ ] Property code/ID is correct
- [ ] Legal ownership confirmed (for listings)
- [ ] Photos/videos are of the correct property

### Content Quality
- [ ] Message is clear and concise
- [ ] Value proposition is evident
- [ ] Call-to-action is specific and compelling
- [ ] No spelling or grammar errors
- [ ] Tone matches brand voice
- [ ] Language appropriate for audience (Thai/English/both)
- [ ] Information is up-to-date and relevant

### Visual Elements
- [ ] High-quality image(s) or video attached
- [ ] Visuals are relevant to the content
- [ ] Correct aspect ratio for platform
- [ ] Branding elements included (logo/watermark if applicable)
- [ ] Text overlays are readable
- [ ] No sensitive/private information visible in background
- [ ] Colors are consistent with brand guidelines

### Engagement Elements
- [ ] Clear call-to-action included
- [ ] Question or conversation starter (if appropriate)
- [ ] Relevant hashtags included
- [ ] @ mentions used correctly
- [ ] Location tag added (if applicable)
- [ ] Link included (where necessary)
- [ ] Contact information provided

### Technical Elements
- [ ] Post scheduled for optimal time
- [ ] Platform selected correctly
- [ ] Post format appropriate (feed/story/reel/video)
- [ ] Character limit respected
- [ ] Emojis display correctly
- [ ] Line breaks formatted properly
- [ ] Preview checked on mobile and desktop

## Platform-Specific Checklists

### Facebook Checklist

#### Before Posting
- [ ] Character count within optimal range (40-80 for engagement, can be longer)
- [ ] 1-3 relevant hashtags included
- [ ] Line breaks used for readability
- [ ] Call-to-action is clear
- [ ] Tagged relevant Pages (if applicable)
- [ ] Location tag added
- [ ] Image size: minimum 1200 x 630 pixels
- [ ] Video: MP4 or MOV format, max 240 minutes
- [ ] Alt text added for accessibility
- [ ] Link preview looks correct

#### Post Settings
- [ ] Audience targeting set (if using ads)
- [ ] Post privacy set correctly (Public recommended)
- [ ] Notifications turned on for engagement monitoring
- [ ] Cross-posting to Instagram configured (if applicable)
- [ ] Schedule time optimized (typically 9 AM - 3 PM weekdays)

#### Facebook-Specific Best Practices
- [ ] First sentence grabs attention (shows before "See More")
- [ ] Native video uploaded (not YouTube link) for better reach
- [ ] Asking a question to encourage comments
- [ ] Link posted as comment (not in post) for better organic reach
- [ ] Story alternative created (if applicable)

---

### Instagram Checklist

#### Before Posting
- [ ] First 125 characters contain key message
- [ ] 10-30 hashtags prepared (in caption or first comment)
- [ ] Image is square (1:1), portrait (4:5), or landscape (1.91:1)
- [ ] Image resolution: minimum 1080 x 1080 pixels
- [ ] Video: MP4 format, 3-60 seconds for feed, up to 90 seconds for Reels
- [ ] Emojis used strategically for visual appeal
- [ ] Line breaks formatted with periods or symbols
- [ ] Location tag added
- [ ] Relevant accounts tagged in image
- [ ] Link in bio updated (if directing traffic)

#### Instagram Feed Post
- [ ] High-quality, visually appealing image
- [ ] Image follows brand aesthetic
- [ ] Caption tells a story or provides value
- [ ] Call-to-action in caption
- [ ] Alt text added for accessibility
- [ ] Multiple images (carousel) used if showing different angles
- [ ] Optimal posting time: 11 AM - 1 PM or 7 PM - 9 PM

#### Instagram Stories
- [ ] Vertical format (9:16 aspect ratio)
- [ ] Resolution: 1080 x 1920 pixels
- [ ] Text readable on mobile devices
- [ ] Stickers/GIFs used appropriately
- [ ] Poll, question, or quiz added for engagement
- [ ] Swipe-up link (if available) or link sticker added
- [ ] Highlights category selected
- [ ] Tagged relevant accounts
- [ ] Location sticker added

#### Instagram Reels
- [ ] Vertical video (9:16 aspect ratio)
- [ ] Duration: 15-90 seconds
- [ ] High-quality video (minimum 1080p)
- [ ] Trending audio used (if appropriate)
- [ ] Text overlays are clear and timed well
- [ ] Hook in first 3 seconds
- [ ] Captions/subtitles added
- [ ] Cover image selected
- [ ] Relevant hashtags in caption
- [ ] Cross-post to Feed toggle checked

## Image/Video Quality Checklist

### Image Quality Standards

#### Technical Requirements
- [ ] Minimum resolution met for platform
- [ ] File format: JPG, PNG, or HEIC
- [ ] File size optimized (under 8 MB recommended)
- [ ] Aspect ratio correct for platform
- [ ] No pixelation or blurriness
- [ ] Colors are vibrant and accurate
- [ ] Proper exposure (not too dark or bright)
- [ ] Sharp focus on subject

#### Composition
- [ ] Subject is clear and prominent
- [ ] Rule of thirds applied (when appropriate)
- [ ] No distracting elements in background
- [ ] Professional staging (for interior shots)
- [ ] Straight horizons/vertical lines
- [ ] Good use of natural light
- [ ] Appropriate depth of field
- [ ] Visual hierarchy guides eye

#### Property-Specific
- [ ] Property features are highlighted
- [ ] Multiple angles shown (for carousel posts)
- [ ] Before/after (for renovations)
- [ ] Key selling points visible
- [ ] Clean and tidy spaces
- [ ] No personal items visible (privacy)
- [ ] License plates blurred (if vehicles visible)
- [ ] People faces blurred (if not consented)

#### Branding
- [ ] Logo/watermark placed consistently
- [ ] Brand colors used
- [ ] Font matches brand guidelines
- [ ] Watermark subtle but visible
- [ ] Contact info included (if applicable)
- [ ] Agency branding on property photos

### Video Quality Standards

#### Technical Requirements
- [ ] Minimum 1080p resolution (1920 x 1080)
- [ ] File format: MP4 (H.264 codec)
- [ ] Frame rate: 30 or 60 fps
- [ ] Aspect ratio correct for platform
- [ ] Audio levels consistent and clear
- [ ] No audio clipping or distortion
- [ ] Subtitles/captions added (accessibility)
- [ ] File size within platform limits
- [ ] Video length appropriate for platform

#### Production Quality
- [ ] Stable footage (tripod or gimbal used)
- [ ] Smooth camera movements
- [ ] Good lighting throughout
- [ ] Clean audio (no background noise)
- [ ] Professional transitions
- [ ] Consistent color grading
- [ ] Intro/outro branded
- [ ] Background music (if used) is royalty-free

#### Content Structure
- [ ] Hook in first 3 seconds
- [ ] Clear narrative or structure
- [ ] Key information highlighted with text
- [ ] Pace keeps viewer engaged
- [ ] Call-to-action at end
- [ ] Logical flow and progression
- [ ] Value delivered throughout
- [ ] Ending encourages action

#### Property Video Specific
- [ ] Virtual tour flow makes sense
- [ ] All key rooms shown
- [ ] Property features highlighted
- [ ] Neighborhood/location featured
- [ ] Agent introduction (if applicable)
- [ ] Property details on-screen
- [ ] Contact information visible
- [ ] Drone footage (if available and adds value)

- [ ] Property availability confirmed
- [ ] No misleading information

- [ ] Permission obtained for all images/videos
- [ ] No copyright infringement

- [ ] Photos are current and accurately represent property

- [ ] No false or misleading claims

# Social Media Caption AI Prompts

## Overview
This document contains AI prompts for generating engaging social media captions for Asset Management Property's content across various platforms. Use these prompts with AI tools like ChatGPT, Claude, or Gemini to create compelling captions that resonate with your target audience.

## How to Use These Prompts

1. **Select the appropriate prompt** based on your platform, target group, and content style
2. **Replace variables in [brackets]** with your specific information
3. **Copy the complete prompt** into your AI tool
4. **Review and customize** the AI-generated output to match your brand voice
5. **Add relevant hashtags** from our hashtag strategy
6. **Test different variations** to find what resonates best with your audience

## Variables Reference Guide

### Property Details Variables
- `[PROJECT_NAME]` - Full project name
- `[PROPERTY_TYPE]` - Condo, Villa, Townhouse, House, Land
- `[LOCATION]` - Specific area in Pattaya (Jomtien, Naklua, Central, etc.)
- `[BEDROOMS]` - Number of bedrooms
- `[BATHROOMS]` - Number of bathrooms
- `[SIZE_SQM]` - Size in square meters
- `[PRICE]` - Price or price range
- `[RENTAL_PRICE]` - Monthly rental price

### Features Variables
- `[FEATURE_1]`, `[FEATURE_2]`, `[FEATURE_3]` - Key property features
- `[VIEW_TYPE]` - Sea view, City view, Pool view, Garden view
- `[AMENITIES]` - Pool, Gym, Security, Parking, etc.
- `[UNIQUE_SELLING_POINT]` - What makes this property special

### Audience Variables
- `[TARGET_AUDIENCE]` - Investors, Families, Retirees, Expats, Digital Nomads
- `[BUYER_MOTIVATION]` - Investment, Lifestyle, Retirement, Income
- `[DEMOGRAPHIC]` - Age range, nationality, income level

### Campaign Variables
- `[OFFER_TYPE]` - Discount, Free furniture, Special financing
- `[DISCOUNT_AMOUNT]` - Percentage or baht amount
- `[VALID_UNTIL]` - Campaign end date
- `[URGENCY_FACTOR]` - Limited units, Last week, Only 3 left

### Tone Variables
- `[TONE]` - Professional, Casual, Urgent, Educational, Inspirational
- `[STYLE]` - Formal, Conversational, Technical, Storytelling
- `[LANGUAGE]` - Thai, English, Both (bilingual)

## Best Practices for AI-Generated Captions

### 1. Always Customize AI Output
- Never post AI-generated captions without review
- Add personal touches and brand voice
- Verify all facts and figures
- Adjust tone for your specific audience

### 2. Maintain Brand Consistency
- Keep Asset Management Property's voice: professional yet approachable
- Use consistent terminology
- Maintain quality standards across all platforms
- Align with overall brand messaging

### 3. Optimize for Each Platform
- Facebook: Longer, more detailed, conversational
- Instagram: Visual, lifestyle-focused, shorter paragraphs
- TikTok: Brief, trendy, hook-driven
- LinkedIn: Professional, data-driven, business-focused
- LINE: Friendly, direct, Thai cultural norms

### 4. Test and Iterate
- Try multiple prompt variations
- A/B test different approaches
- Track engagement metrics
- Refine based on performance

### 5. Add Human Element
- Include real experiences
- Use actual property photos
- Add personal insights
- Maintain authenticity

### 6. Hashtag Strategy
- Generate caption first
- Add hashtags separately
- Use platform-appropriate number
- Mix branded and trending hashtags

### 7. Call-to-Action Guidelines
- Make it clear and specific
- Match CTA to platform
- Create appropriate urgency
- Multiple contact options when relevant

### 8. Cultural Sensitivity
- Respect Thai cultural norms
- Use appropriate language for Thai audience
- Consider Thai holidays and seasons
- Adapt humor and tone culturally

### 9. Legal Compliance
- Verify all claims
- Don't promise guaranteed returns
- Follow Thai real estate advertising laws
- Include required disclosures

### 10. Timing Optimization
- Generate captions in advance
- Schedule for optimal posting times
- Prepare variations for different times
- Plan around Thai holidays and events

## Troubleshooting AI Caption Generation

### If captions are too generic:
- Add more specific variables
- Include unique property details
- Specify target audience more narrowly
- Request multiple variations

### If tone is off:
- Be more specific about desired tone
- Provide example captions you like
- Specify what to avoid
- Use emotion-specific keywords

### If too long/short:
- Specify character count explicitly
- Request specific sections to expand/cut
- Provide structure template
- Ask for multiple length options

### If not platform-appropriate:
- Reference platform best practices in prompt
- Provide platform-specific examples
- Mention platform culture
- Request platform-native language

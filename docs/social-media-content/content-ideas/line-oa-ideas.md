# 💚 LINE Official Account Content Ideas - Asset Management Property

> กลยุทธ์คอนเทนต์สำหรับ LINE Official Account

## 🎯 LINE OA Strategy Overview

### Platform Characteristics
- **Audience:** All ages, but especially 25-55 in Thailand
- **Content Lifespan:** Push messages have immediate impact
- **Best For:** Direct communication, promotions, customer service, conversions
- **Format:** Text, Rich Messages, Rich Menus, Flex Messages, Carousels

### Broadcast Frequency
- **2-3 broadcasts/week** (Don't overdo it!)
- **Best times:** 10:00 AM, 6:00 PM
- **Chatbot:** 24/7 automated responses

## 🎨 Design Best Practices

### Rich Message Guidelines

**Images:**
- Size: 1040 x 1040 px (1:1) or 1040 x 1300 px
- File size: < 1 MB
- High quality, clear text
- Mobile-optimized

**Colors:**
- Use brand colors
- High contrast for readability
- Eye-catching but not overwhelming

**Text:**
- Large enough to read on mobile
- Thai font that renders well
- Key info above the fold
- Clear CTA buttons

### Flex Message Best Practices

- Use LINE's official Flex Message Simulator
- Test on different screen sizes
- Keep it simple and scannable
- Maximum 10 bubbles per message

## 📊 Performance Metrics

### LINE OA KPIs

| Metric | Target | Note |
|--------|--------|------|
| Open Rate | >40% | Broadcast messages |
| Click-through Rate | >10% | Link clicks |
| Response Rate | >20% | User replies |
| Friend Growth | +100/week | New followers |
| Message Replies | <30 min | Average response time |
| Block Rate | <5% | Keep low! |

### Content Performance Tracking

Track which content types get:
- Highest open rates
- Most replies
- Best conversions
- Lowest blocks

Adjust strategy accordingly.

## 🚫 Things to Avoid

❌ **Don't:**
- Broadcast more than 3x/week
- Send messages late at night
- Use all caps
- Be too promotional
- Spam with same content
- Ignore user messages
- Use low-quality images

✅ **Do:**
- Provide value
- Be timely and relevant
- Personalize when possible
- Respond quickly
- Test broadcast times
- Segment audience if possible
- Track and optimize

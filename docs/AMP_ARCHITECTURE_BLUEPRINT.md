|-------|------------|---------|---------|

| Component | Technology | Purpose |
|-----------|------------|---------|

| Component | Technology | Purpose |
|-----------|------------|---------|

| **Queue** | Redis | Message queue |
| **Task Runner** | Celery | Async tasks |

|------|------------|----------|

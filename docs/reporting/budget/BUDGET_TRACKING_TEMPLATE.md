## Usage Guidelines

### Daily Budget Monitoring

**Morning (9:00 AM):**
1. Check yesterday's ad spend
2. Compare to daily budget target
3. Review any overspend alerts
4. Adjust bids if needed

**Evening (5:00 PM):**
1. Record final daily spend
2. Calculate daily CPL
3. Note any anomalies
4. Plan tomorrow's allocation

### Weekly Budget Review

**Every Monday:**
1. Review last week's spend
2. Calculate week-over-week variance
3. Analyze CPL by source
4. Adjust weekly allocations
5. Update forecast if needed

### Monthly Budget Close

**Last Day of Month:**
1. Finalize all expenses
2. Calculate total variance
3. Analyze ROI by channel
4. Prepare monthly report
5. Plan next month's budget
